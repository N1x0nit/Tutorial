Как загрузить сайт на GitHub

Следуй простым шагам, чтобы загрузить свой сайт на GitHub и получить ссылку для доступа.

Шаги для загрузки вашего сайта:

  1. Создай репозиторий на GitHub. Перейди на главную страницу GitHub, нажимаешь Create Rpository
  2. Настрой репозиторий. В строке Repository name вводишь свое название репозитория, Description не трогаешь, ниже ставишь Public, потом ставишь галочку Add a README file и в конце Create repository
  3. Добавь файлы сайта в репозиторий. После всех действий у тебя откроется твой репозиторий где ты должен будешь нажать uploading an existing file, после этого перенеси index.html, styles.css и script.js в папку репозитория и нажми Commit changes.
  4. Настройте GitHub Pages. В настройках репозитория выберите ветку main в разделе GitHub Pages и нажми save.

После выполнения этих шагов подожди от 40 до 80 секунд и твой сайт будет доступен по следующему адресу:

"https://ваше-имя-пользователя.github.io/название-репозитория"
